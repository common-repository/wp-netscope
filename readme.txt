Netscope is a web-based analytics suite. This plugin installs the necessary tags on your WordPress site.

== Description ==

[Netscope](http://netscope.marktest.pt/) is a web-based analytics suite.

== Installation ==

= Automatic =

1. In the admin panel plugins page click Add New
2. Search for WP Netscope
3. Find WP Netscope in the list an click Install Now
4. Click OK when prompted
5. Activate the plugin
6. Done!

= Manual =

1. Extract wp-netscope.zip to your `wp-content/plugins` directory.
2. In the admin panel under plugins activate WP Netscope.
3. Done!

= Configuration =

1. Visit the newly created entry for WP Netscope on your dashboard's Settings menu
3. Save your settings
